# 4k下落式音游谱面生成器开发文档

## 1. 项目目标

实现一个自动谱面生成工具，具备以下能力：

-  输入：MP3 音频文件 
-  输出：符合指定数据结构的音乐游戏谱面（Note Chart） 
-  功能： 
  -  自动分析 BPM 
  -  自动分段（主歌 / 副歌 / 过渡 / 静音等） 
  -  情绪与能量检测（强 / 弱） 
  -  根据难度参数生成不同密度的谱面 
-  提供简单 GUI 工具： 
  -  导入音频 
  -  设置难度 
  -  点击生成 
  -  导出谱面 

## 2. 技术选型

### 核心技术栈

-  音频分析：`librosa`
-  数值计算：`numpy`
-  GUI：`customtkinter`
-  文件处理：`soundfile` / `pydub`
-  可选优化： 
  - `scipy`（信号处理） 
  - `madmom`（节拍检测更强，但复杂） 

## 3. 系统架构

[GUI层]

  └── 音频导入 / 参数设置 / 触发生成

[控制层]

  └── ChartGenerator（主流程控制）

[分析层]

  ├── BeatAnalyzer（节拍/BPM）

  ├── StructureAnalyzer（段落分析）

  ├── EnergyAnalyzer（能量/情绪）

[生成层]

  └── NoteMapper（谱面生成）

[输出层]

  └── ChartSerializer（导出JSON）

## 4. 音频分析模块设计（核心）

### 4.1 BPM检测

使用 `librosa.beat.beat_track`

tempo, beat_frames = librosa.beat.beat_track(y=y, sr=sr)

beat_times = librosa.frames_to_time(beat_frames, sr=sr)

输出：

- `tempo`（BPM） 
- `beat_times`（每拍时间点） 

### 4.2 Onset检测（打击点）

用于决定音符生成位置：

onset_frames = librosa.onset.onset_detect(y=y, sr=sr)

onset_times = librosa.frames_to_time(onset_frames, sr=sr)

说明：

-  onset ≈ 音乐中的“可打点” 
-  是生成音符的基础 

### 4.3 能量分析（情绪/强度）

核心指标：RMS（音量能量）

rms = librosa.feature.rms(y=y)[0]

times = librosa.frames_to_time(range(len(rms)), sr=sr)

处理方式：

-  归一化 RMS 
-  计算滑动平均 
-  划分能量区间： 

| 能量值  | 含义           |
| ------- | -------------- |
| 0.0–0.3 | 安静（休息段） |
| 0.3–0.6 | 中等           |
| 0.6–1.0 | 激烈（高潮段） |

### 4.4 频谱特征（辅助情绪判断）

使用：

spectral_centroid = librosa.feature.spectral_centroid(y=y, sr=sr)

意义：

-  高频多 → 明亮 / 激烈 
-  低频多 → 稳重 / 缓慢 

### 4.5 段落划分（重点难点）

方法一（推荐）：自相似矩阵 + 聚类

import librosa.segment

C = librosa.feature.chroma_cqt(y=y, sr=sr)

recurrence = librosa.segment.recurrence_matrix(C)

然后：

-  使用 `librosa.segment.agglomerative` 聚类 
-  得到段落边界 

### 简化方案（工程可落地）

基于能量 + 节拍密度：

if rms_mean > threshold_high:

​    segment = "高潮"

elif rms_mean < threshold_low:

​    segment = "休息"

else:

​    segment = "过渡"

## 5. 谱面生成算法

### 5.1 基础规则

输入：

- `beat_times`
- `onset_times`
- `energy_curve`
- `difficulty`

输出：

-  Note List 

### 5.2 音符生成逻辑

#### Step 1：候选点

候选点 = onset_times ∪ beat_times

#### Step 2：根据难度筛选

density = difficulty  # 0~1

selected_notes = []

for t in candidates:

​    if random.random() < density:

​        selected_notes.append(t)

#### Step 3：根据能量调整密度

final_density = base_density * (1 + energy * 0.5)

效果：

-  高潮 → 更密集 
-  安静 → 更稀疏 

#### Step 4：轨道分配（4K）

lane = random.randint(0, 3)

进阶策略：

-  避免连续同轨 
-  高频 → 多手切换 
-  低频 → 稳定节奏 

### 5.3 长按音（Hold Note）

检测条件：

-  持续高能量段 
-  或 onset 间隔较大 

if next_onset - current_onset > threshold:

​    create_hold_note()

## 7. 难度系统设计

参数：`difficulty ∈ [0, 1]`

影响：

| 维度                  | 作用 |
| --------------------- | ---- |
| note密度              | ↑    |
| 同时按键概率          | ↑    |
| 长按比例              | ↑    |
| 节拍细分（1/4 → 1/8） | ↑    |
